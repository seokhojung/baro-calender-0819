## 🎯 스토리 목표

바로캘린더 애플리케이션의 **다양한 컴포넌트 스타일링 패턴**을 구현하여 개발팀이 일관되고 효율적인 방법으로 UI 컴포넌트를 스타일링할 수 있는 통합된 시스템을 제공한다.

### 주요 구현 사항
- **Tailwind CSS 확장** 설정으로 디자인 토큰과 통합
- **CSS Modules** 패턴으로 스코프된 스타일링
- **Class Variance Authority (CVA)** 기반 변형 관리
- **조건부 스타일링** 및 **CSS-in-JS 최적화** 패턴

---

## 👤 사용자 스토리

### US 2.15.1: Tailwind CSS 확장 시스템
**As a** 개발팀원  
**I want** 디자인 토큰과 완전히 통합된 Tailwind CSS 설정을 사용하기를  
**So that** 유틸리티 클래스와 커스텀 디자인 시스템이 일관되게 작동하며 개발 속도를 높일 수 있다.

**인수 조건:**
- [ ] 프로젝트 색상, 간격, 그림자 토큰의 Tailwind 통합
- [ ] 커스텀 애니메이션 및 키프레임 정의
- [ ] 다크모드 자동 감지 및 클래스 기반 테마 전환
- [ ] Tailwind 플러그인 통합 (forms, typography, aspect-ratio)

### US 2.15.2: 모듈식 CSS 패턴
**As a** 개발팀원  
**I want** CSS Modules와 CVA를 통한 체계적인 스타일 관리 시스템을 사용하기를  
**So that** 스타일 충돌 없이 재사용 가능한 컴포넌트를 만들고 변형을 쉽게 관리할 수 있다.

**인수 조건:**
- [ ] CSS Modules 기반 스코프된 스타일링 구현
- [ ] CVA를 통한 변형(variant) 및 크기(size) 관리
- [ ] 데이터 속성 기반 조건부 스타일링
- [ ] 복합 컴포넌트(Card, Button 등) CVA 패턴 적용

### US 2.15.3: 성능 최적화된 CSS-in-JS
**As a** 개발팀원  
**I want** 최적화된 CSS-in-JS 패턴을 통해 런타임 성능을 보장하기를  
**So that** 동적 스타일링이 필요한 경우에도 애플리케이션 성능에 영향을 주지 않을 수 있다.

**인수 조건:**
- [ ] Emotion 기반 정적/동적 스타일 분리
- [ ] 스타일 컴포지션 패턴으로 재사용성 향상
- [ ] 테마별 스타일 생성 최적화
- [ ] 조건부 스타일링 성능 최적화

      <div className={styles.calendarHeader}>
        <h2 className={styles.calendarTitle}>
          {format(currentDate, 'yyyy년 M월', { locale: ko })}
        </h2>
        
        <div className={styles.navigationButtons}>
          <button
            className={styles.navigationButton}
            onClick={handlePreviousMonth}
            aria-label="이전 달"
          >
            <ChevronLeftIcon className="w-5 h-5" />
          </button>
          
          <button
            className={styles.navigationButton}
            onClick={handleNextMonth}
            aria-label="다음 달"
          >
            <ChevronRightIcon className="w-5 h-5" />
          </button>
        </div>
      </div>
      
      {/* 캘린더 그리드 */}
      <div className={styles.calendarGrid}>
        {/* 요일 헤더 */}
        {weekdays.map((weekday) => (
          <div key={weekday} className={styles.weekdayHeader}>
            {weekday}
          </div>
        ))}
        
        {/* 날짜 셀들 */}
        {calendarDays.map((date) => {
          const dateKey = format(date, 'yyyy-MM-dd')
          const dayEvents = eventsByDate[dateKey] || []
          const isSelected = selectedDate && isSameDay(date, selectedDate)
          const isTodayDate = isToday(date)
          const isCurrentMonth = isSameMonth(date, currentDate)
          
          return (
            <button
              key={dateKey}
              className={styles.dayCell}
              data-selected={isSelected}
              data-today={isTodayDate}
              data-other-month={!isCurrentMonth}
              data-has-events={dayEvents.length > 0}
              onClick={() => handleDateClick(date)}
              aria-label={`${format(date, 'M월 d일', { locale: ko })}${dayEvents.length > 0 ? `, ${dayEvents.length}개 이벤트` : ''}`}
            >
              <span className={styles.dayNumber}>
                {date.getDate()}
              </span>
              
              {dayEvents.length > 0 && (
                <div className={styles.eventList}>
                  {dayEvents.slice(0, 3).map((event) => (
                    <div
                      key={event.id}
                      className={styles.eventItem}
                      data-project={getProjectColor(event.projectId)}
                      onClick={(e) => {
                        e.stopPropagation()
                        onEventClick?.(event)
                      }}
                      title={event.title}
                    >
                      {event.title}
                    </div>
                  ))}
                  {dayEvents.length > 3 && (
                    <div className={cn(styles.eventItem, 'text-muted-foreground')}>
                      +{dayEvents.length - 3}개 더
                    </div>
                  )}
                </div>
              )}
            </button>
          )
        })}
      </div>
    </div>
  )
